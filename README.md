# swift-gomamayo

Gomamayo detection available on Apple platforms.

```console
$ swift run gomamayo ゴママヨ
1 ary 1 degree(s) gomamayo is found
```

Gomamayo (ゴママヨ) is a kind of Japanese wordplay.
